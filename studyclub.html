Study Club

Kegiatan diskusi internal sesama anggota KSM Android. Disana membahas tentang teknologi Pemrograman Android dan Web. Study Club berbentuk kelas, tiap kelas beda level pengajarannya yaitu:

UI/UX Basic

Health Everyday
My Skin
Masakpedia

UI UX Basic adalah kelompok studi peminatan yang mempelajari User Interface dan User Experience dari sebuah aplikasi, seperti melakukan research, merancang userflow, serta mendesain prototype dari aplikasi yang akan dibangun. Dengan memanfaatkan tools design seperti Figma, Adobe XD, dan lainnya.

Android

Android Basic

Android Basic adalah kelompok studi peminatan yang mempelajari dasar-dasar Pemrogramman Android menggunakan bahasa pemrogramman Kotlin.

Android Beginner

Android Beginner adalah kelompok studi peminatan yang mengimplementasikan pembelajaran yang telah didapatkan dari kelas java basic dalam pemebntukan aplikasi android serta membuat layout dan hal teknis lainnya.

Website

Web Basic

Web Basic adalah kelompok studi peminatan yang mempelajari dasar-dasar Pemrogramman Web dengan materi fundamental pengembangan web, seperti pengenalan front-end, menyusun layout dan struktur website dengan bahasa HTML, CSS, serta JavaScript.

Web Beginner

Web Beginner adalah kelompok studi peminatan yang mempelajari Pemrograman Web dengan tingkatan yang lebih mendalam, seperti membuat CRUD dengan bahasa PHP dan MySQL.

Backend

Backend Basic

Backend Basic adalah kelompok studi peminatan yang mempelajari hal yang berhubungan dengan server (misal pada situs atau cloud) dan database secara kompleks.
